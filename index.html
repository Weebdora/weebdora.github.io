Let's get started.

Doctype declaration. The first line of code required in every HTML document.The DOCTYPE declaration is an instruction to the web browser about what version of HTML the page is written in. This ensures that the web page is parsed the same way by different web browsers.

Html element. It represents the root level element and wraps all the content on the page. All other elements must be descendants of this element

Head element. It acts as a container for everything you want to include on the HTML page, that isn't the content the page will show to viewers. This includes keywords and a page description that would appear in search results, CSS to style content, JavaScript to add interactivity, character set declarations, and more.

Body element. It contains all the content that displays on the page, including text, images, videos, games, playable audio tracks, or whatever else.

Meta charset="utf-8". This element specifies the character set for your document to UTF-8

Title element.This sets the title of the page and is also used to describe the page when it is bookmarked
